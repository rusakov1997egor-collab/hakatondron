# Project State - Hackathon Drone CV System

**Дата последнего обновления:** 2026-04-27

## Суть проекта
Хакатон-проект для анализа видеопотока с телефона (имитация дрона). Система в реальном времени:
- Детектирует помехи (птицы, деревья, другие дроны) с помощью YOLOv8
- Распознает жест "иди ко мне" с помощью MediaPipe

## Технологический стек
- **OpenCV** - работа с видеопотоком (RTSP/HTTP)
- **YOLOv8 (Ultralytics)** - детекция объектов
- **MediaPipe** - распознавание жестов
- **Python 3.x** - основной язык
- **Threading** - многопоточность для стабильного чтения потока

## Готовые модули

### ✅ requirements.txt
Все зависимости: opencv-python, ultralytics, mediapipe, numpy

### ✅ src/stream_reader.py
**Класс:** `StreamReader`
- Чтение RTSP/HTTP потока в отдельном потоке (threading)
- Автоматическое переподключение при потере связи
- Потокобезопасное получение кадров через lock
- Минимальная задержка (буфер = 1 кадр)

**Методы:**
- `start()` - запуск потока
- `stop()` - остановка
- `read()` - получение последнего кадра
- `is_available()` - проверка доступности кадров

### ✅ src/object_detector.py
**Класс:** `ObjectDetector`
- Обертка над YOLOv8n (легковесная модель)
- Детекция помех: bird, airplane, person
- Отрисовка bounding boxes с подписями

**Методы:**
- `process_frame(frame)` - возвращает (кадр с рамками, список помех)
- `get_obstacle_classes()` - список отслеживаемых классов

**Настройки:**
- Confidence threshold: 0.5
- Цвет рамок: красный (0, 0, 255)
- Легко изменяемый список OBSTACLE_CLASSES

## В разработке

### 🔄 src/gesture_reader.py
**Планируется:**
- Класс-обертка для MediaPipe Hands
- Детекция жеста "иди ко мне" (отслеживание координат пальцев)
- Логика распознавания изменения позиции руки

### 🔄 main.py
**Планируется:**
- Точка входа с бесконечным циклом `while True`
- Инициализация StreamReader, ObjectDetector, GestureReader
- Обработка кадров в реальном времени
- Вывод результатов (детекции + жесты)
