# DronNav - Гибридная логистическая система доставки

Система симуляции доставки грузов с использованием связки грузовик-дрон, разработанная для хакатона. Проект демонстрирует концепцию TSP-D (Traveling Salesman Problem with Drone) с управлением жестами и визуализацией в реальном времени.

## Архитектура

Проект состоит из трех основных компонентов:

### 1. Backend (FastAPI)
- **Путь**: `backend/`
- **Технологии**: FastAPI, WebSocket, Pydantic
- **Функции**:
  - REST API для управления доставкой
  - WebSocket для телеметрии в реальном времени
  - Физический движок дрона (скорость, ускорение, расход батареи)
  - A* навигация с обходом препятствий
  - Симуляция полного цикла: Склад → ПВЗ → возврат

**Основные эндпоинты**:
- `POST /start` - запуск доставки
- `POST /gesture` - отправка жеста управления
- `GET /status` - текущий статус системы
- `WS /ws/telemetry` - поток телеметрии

### 2. Frontend (Next.js 16)
- **Путь**: `frontend/`
- **Технологии**: Next.js 16.2.4, React 19, TypeScript, Tailwind CSS 4
- **Функции**:
  - Визуализация карты с препятствиями (45 домов)
  - Отображение грузовика, дрона, склада и ПВЗ
  - Телеметрия: YAW, расстояние, время полета
  - Индикатор заряда батареи
  - Управление через жесты

### 3. Gesture Recognition (MediaPipe)
- **Путь**: `gesture_recognition/`
- **Технологии**: OpenCV, MediaPipe, NumPy
- **Функции**:
  - Распознавание жеста "открытая ладонь → кулак"
  - Отправка команды "coming" на backend
  - Автоматическая загрузка модели MediaPipe

## Установка и запуск

### Требования
- Python 3.8+
- Node.js 18+
- Веб-камера (для распознавания жестов)

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Откройте http://localhost:3000

### Gesture Recognition

```bash
cd gesture_recognition
pip install opencv-python mediapipe numpy requests
python gesture_reader.py
```

При первом запуске автоматически скачается модель `hand_landmarker.task`.

## Как использовать

1. Запустите backend и frontend
2. Откройте веб-интерфейс
3. (Опционально) Запустите gesture_reader.py для управления жестами
4. Нажмите "Старт доставки" или покажите жест "открытая ладонь → кулак"
5. Наблюдайте за процессом доставки в реальном времени

## Логика работы

1. **Старт**: Дрон стартует со склада (левый верхний угол)
2. **Навигация**: A* алгоритм строит путь к ПВЗ, обходя препятствия
3. **Физика**: Реалистичная симуляция полета с ускорением и расходом батареи
4. **Доставка**: Дрон достигает ПВЗ (правый нижний угол)
5. **Возврат**: Автоматический возврат на склад

## Технические детали

### Физика дрона
- Максимальная скорость: 5 м/с
- Ускорение: 2 м/с²
- Расход батареи: 0.5% на метр
- Начальный заряд: 100%

### Навигация
- Сетка: 200×200 метров
- Размер ячейки: 2 метра
- Алгоритм: A* с эвристикой Евклида
- Препятствия: 45 зданий различной формы

### WebSocket телеметрия
Обновления каждые 100ms:
- Позиция дрона (x, y)
- Угол поворота (yaw)
- Расстояние до цели
- Время полета
- Заряд батареи
- Статус миссии

## Разработка

### Правила архитектуры
- Backend - единственный источник истины для физики и логики
- Frontend - тонкий клиент, только визуализация
- Никакой физики или бизнес-логики в React компонентах
- WebSocket для real-time данных, REST для команд
